# 📞 Telecom Call Intent Classifier (ML + Hybrid AI)

## 🚀 Overview

This project builds an AI system to classify customer queries into categories like Billing, Repair, Retention, Sales, and Order Management.

## 🧠 Approach

* TF-IDF Vectorization
* Logistic Regression Model
* Class imbalance handling
* Rule-based override system

## 📊 Performance

* Accuracy: ~91%
* Strong performance across all classes

## ⚠️ Limitation

Traditional ML struggles with implicit queries like:

* "I might switch provider"
* "This service is frustrating"

## 🔥 Solution Upgrade

Designed a **Hybrid AI System**:

* Rule-based intent detection
* ML classifier
* (Planned) LLM fallback for complex queries

## 🛠 Tech Stack

* Python
* Scikit-learn
* Pandas
* NLP
